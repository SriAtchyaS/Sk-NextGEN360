🚀 SETUP STEPS

Step 1: Install New Dependencies
─────────────────────────────────────────────────────────────
   cd python_app
   pip install -r requirements.txt

   New packages:
   - openpyxl (Excel file reading)
   - httpx (Gemini API)


Step 2: Apply Database Schema
─────────────────────────────────────────────────────────────
   Option A - Direct SQL:
   psql -h hackathon2.cluster-ctokk08w4mh5.ap-south-1.rds.amazonaws.com \
        -U bug_makers -d bug_makers \
        -f python_app/database_schema.sql

   Option B - Copy & paste SQL from:
   python_app/database_schema.sql


Step 3: Start Backend
─────────────────────────────────────────────────────────────
   cd python_app
   python run.py

   ✓ Wait for: "Application starting on port 5000"


Step 4: Start Frontend
─────────────────────────────────────────────────────────────
   Open NEW terminal:
   cd Frontend
   npm start

   ✓ Browser opens at: http://localhost:3000

🔧 NEW API ENDPOINTS

Manager:
─────────────────────────────────────────────────────────────
POST   /api/manager/assign-task          Upload Excel + assign
GET    /api/manager/assigned-tasks       Get all tasks
GET    /api/manager/my-freshers          Get freshers (updated)
POST   /api/mock-test/generate-for-task  AI generates 20 questions

Fresher:
─────────────────────────────────────────────────────────────
GET    /api/fresher/my-tasks             Get tasks + subtopics
POST   /api/fresher/complete-task/{id}   Mark completed
GET    /api/mock-test/start-by-task/{id} Start test by task ID
POST   /api/mock-test/submit             Submit answers

Rules:
- First column only (Column A)

- Any number of rows
